# LEXIE

LEXIE is a modular voice AI platform.

It turns speech into intelligence, then intelligence back into speech.

Pipeline:

User Voice
-> STT
-> LLM Agent
-> TTS
-> Voice Response

## Goals

- Replace any provider easily
- Build multiple intelligent agents
- Keep architecture clean
- Support future Mac / mobile 

## Repository Structure

LEXIE/
- packages/stt
- packages/llm
- packages/tts
- shared
