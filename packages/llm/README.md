# LLM Package

## Purpose

Core intelligence engine of LEXIE.

Transforms text input into useful responses, plans, and actions.

## Responsibilities

- understand user intent
- generate responses
- run agent behavior
- manage personas
- support tools later
- support memory later

## Non Responsibilities

- speech recognition
- audio synthesis
- device microphone handling

## Long-Term Goal

Become a reusable platform for many LEXIE agents.
